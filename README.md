# MineLess
Minecraft Nukkit Server to Play Minning;

##nukkit 광산 관리 플러그인
```
한글광산 관리 플러그인
01. /광산 <추가모드, 제거모드, 일반모드, 확률설정>
 - 추가모드
  -permission : op
  -usage: 명령어 입력 후 "일반모드"가 되기전까지 터치한 블럭들이 광산으로 변한다
 - 제거모드
  -permission: op
  -usage: 명령어 입력 후 "일반모드"가 되기 전 까지 터치한 블럭들이 일반블럭으로 변한다
 - 일반모드
  -permission: op
  -usage: 명령어 입력 시 "추가모드"혹은 "제거모드"에서 벗어난다(아무블럭이나 터치해도 무방)

02. /광산 확률조정 <목록 : string>
 - 목록
  - permission: op
  - usage: 현재 광산에서 리젠되는 광물목록과 확률을 볼 수 있습니다

03. /광산 확률조정 <조정 : string> <확률 : int>
 - 조정
  - permission: op
  - usage: 손에 든 블럭을 광산에 등록하거나, 삭제하거나, 확률을 수정합니다
   {
    확률이 0일경우 : 광산에서 나오는 블럭 삭제

    확률이 0~1000일 경우 :
     - 이미 광산에 있는 블럭 : 리젠확률이 수정된다
     - 광산에 없는 블럭일 경우 : 광산에 추가된다
   }

04. 광산 채굴 시
 - 광산에서 채굴하는 경우 블럭이 엔디티로 떨어지지않고 유저의 인벤토리로 들어갑니다.
 - 곡괭이의 등급에 맞는 광물만 캘 수 있습니다
  example) 돌곡괭이 -> 금원석(x)
   다이아 곡괭이 -> 철원석 (o)
```
